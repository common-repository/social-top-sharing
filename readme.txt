== Description ==

Top Social sharing is a Wordpress plugin of easy use that lets you view the most popular social buttons (Facebook, Twitter, Google Plus) into your pages. And it's possible to customize the features of each of them in a simple and immediate way.

== Installation ==

1. Click on "Plugin" -> "Add new";
2. Click on "Upload media";
3. Activate plugin;
4. After installing click on "Settings" -> "Social top sharing" to edit the plugins options;
